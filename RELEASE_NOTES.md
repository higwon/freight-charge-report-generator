# 릴리즈 노트

## v0.8.0

- Source 파일의 한국어 헤더를 자동 인식하도록 추가했습니다.
- `JOB일자`, `기능코드`, `출발지포트`, `도착지포트`, `매출매입구분`, `거래처코드`, `거래처명`, `현지금액` 헤더를 기존 영문 표준 컬럼으로 매핑합니다.
- 한국어 헤더가 포함된 Excel 2003 XML `.xls` 파일도 Source 시트 자동 탐지와 AR/AP 월별 보고서 생성이 가능하도록 개선했습니다.

## v0.7.0

- AR/AP 월별 보고서의 `차이` 용어를 `마진`으로 변경했습니다.
- 코드별 상세 표를 거래처 중심에서 항구 중심 구조로 변경했습니다.
- 항구 요약행을 기본으로 표시하고, 펼치면 해당 항구의 거래처별 상세 금액을 확인할 수 있도록 Excel 아웃라인을 적용했습니다.
- Overview와 코드별 시트에서 Customer 랭킹을 제거하고 매출·매입 상위 Port만 표시합니다.
- Overview에서 매출만발생/매입만발생 카운트와 검토 대상 표를 제거했습니다.
- 코드별 시트의 월별 컬럼 아웃라인을 제거하고 항구/거래처 행 아웃라인만 유지합니다.
- 숫자는 우측 정렬, 한글 텍스트와 마진율/상태값은 가운데 정렬되도록 정리했습니다.

## v0.6.0

- Overview의 상위 항목을 매출 상위 Port, 매입 상위 Port, 매출 상위 Customer, 매입 상위 Customer로 분리했습니다.
- 각 물류코드 시트에 누적 금액 기준 매출·매입 Top 5를 추가했습니다.
- 항구 기준이 없는 물류코드는 매출·매입 Customer Top 5만 표시합니다.
- 코드별 상세 표의 중복 헤더를 `기본정보`, `누적 합계`, 월별 그룹 구조로 정리했습니다.

## v0.5.0

- AR/AP 월별 보고서에 `Overview` 시트를 추가했습니다.
- 월별보고서 Overview에 기존 요약 Overview의 주요 항목인 메타정보, 코드별 요약, 월별 요약, 상위 Port/Customer, 차트를 포함했습니다.
- Overview에 전체 KPI, 검토 대상, 차이 상위/하위 거래처를 추가했습니다.
- 코드별 시트 상단에 총 누적 및 월별 매출계, 매입계, 차이, 마진율 요약을 추가했습니다.
- 코드별 상세 행에 누적 매출계, 누적 매입계, 누적 차이, 누적 마진율 컬럼을 추가했습니다.
- 마진율 표시를 `-`, `매출만발생`, `매입만발생`, 정상 마진율로 구분했습니다.
- 월별 상세 컬럼을 Excel 그룹으로 접고 펼칠 수 있도록 했습니다.
- 숫자는 불필요한 소수점이 보이지 않도록 `#,##0.##` 형식을 유지합니다.

## v0.4.0

- `AR/AP 월별 보고서` 형식을 추가했습니다.
- 기본 선택 보고서를 `AR/AP 월별 보고서`로 변경했습니다.
- 기존 기본 보고서와 분석 보고서를 `요약 보고서`로 통합했습니다.
- 요약 보고서의 시트 순서를 `Source`, `Overview`, `ALL`, 코드별 시트 순서로 변경했습니다.
- 결과 파일의 첫 번째 시트에 원본 데이터를 `Source`로 유지합니다.
- 물류코드별로 별도 시트를 생성합니다.
- 수출 코드 `AE`, `OE`, `OA`는 도착항과 거래처 기준으로 집계합니다.
- 수입 코드 `AI`, `OI`는 출발항과 거래처 기준으로 집계합니다.
- 기타 물류코드는 항구 없이 거래처 기준으로 집계합니다.
- 월별 `AR계`, `AP계`, `차이`, `마진율` 컬럼을 생성합니다.
- 진행 메시지와 결과 요약 문구를 새 보고서 형식에 맞게 정리했습니다.

## v0.3.0

- UI에서 기본 보고서와 분석 보고서를 선택할 수 있습니다.
- 기존 기본 보고서 형식과 시트 순서를 유지합니다.
- 분석 보고서에 Overview 시트와 Func Code 단계가 포함된 ALL 시트를 추가했습니다.
- Overview에 Func Code별, 월별 합계, 상위 Port/Customer 및 차트를 제공합니다.
- 분석 보고서의 Source 시트에 자동 필터를 적용합니다.

## v0.2.1

- XML 형식 Excel 파일에 이스케이프되지 않은 `&` 문자가 있어도 자동 보정해 처리합니다.
- Source Excel 파일을 애플리케이션 창에 드래그 앤 드롭할 수 있습니다.

## v0.2.0

- 고정 시트명 대신 필수 컬럼 헤더를 기준으로 Source 시트를 자동 탐지합니다.
- 원본 데이터 시트명을 `RawData`에서 `Source`로 변경했습니다.
- AE, AI, OA, OE, OI를 통합한 `ALL` 요약 시트를 추가했습니다.
- Month와 Port를 접고 펼칠 수 있는 Excel 그룹 기능을 추가했습니다.
- 앱 아이콘, 타이틀 이미지, 버전 및 제작자 정보를 적용했습니다.
- 성공/실패 상태 이미지를 결과 요약 영역에 표시합니다.
- 긴 결과 경로와 오류 내용을 확인할 수 있도록 결과 요약 스크롤을 추가했습니다.
- 작업 단계를 더 명확하게 표시하고 버튼 문구를 한국어로 정리했습니다.
- 금액의 소수점은 필요한 경우에만 표시합니다.

## v0.1.0

- 운임 요약 보고서 생성 데스크톱 애플리케이션 최초 버전입니다.
- Excel 2003 XML Spreadsheet, `.xls`, `.xlsx` 파일을 지원합니다.
- 원본 데이터와 Func Code별 요약 시트를 생성합니다.
- 피벗 스타일 서식과 선택적 소수점 표시를 적용합니다.
- 진행 상태, 결과 요약, 결과 파일 및 폴더 열기 기능을 제공합니다.
